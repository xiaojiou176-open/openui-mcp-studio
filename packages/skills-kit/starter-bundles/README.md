# Starter Bundles

These bundles turn the public starter pack into something closer to a real
distribution shelf:

- sample config
- skill bundle
- proof loop
- troubleshooting
- honest official-surface boundary

Current bundle set:

- `codex.mcp.json`
  - quickest Codex stdio MCP config snippet
- `claude-code.mcp.json`
  - quickest Claude Code stdio MCP config snippet
- `openclaw.mcp.json`
  - quickest OpenClaw-style stdio bridge snippet
- `codex-plugin/`
  - Codex Plugin Directory-ready bundle
- `claude-code-plugin/`
  - Claude Code marketplace-ready bundle
- `openclaw-skill/`
  - OpenClaw / ClawHub-ready skill bundle

These are repo-owned artifacts only.
They do not claim publication, approval, or listing inside any official public
catalog.
