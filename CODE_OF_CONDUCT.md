# Code of Conduct

## Our Standard

We want this repository to be a place where contributors can collaborate
without hostility, ridicule, or intimidation.

Examples of behavior that help the project:

- giving technical feedback respectfully
- asking clarifying questions without sarcasm
- documenting risks and trade-offs honestly
- keeping security-sensitive discussion private when appropriate
- helping others understand context instead of gatekeeping it

Examples of unacceptable behavior:

- personal attacks or harassment
- discriminatory or exclusionary language
- sharing secrets, private infrastructure details, or exploit material in
  public spaces
- dismissing contributors for asking basic questions
- intentionally misrepresenting test, CI, or security status

## Scope

This code of conduct applies to:

- repository issues and pull requests
- discussions tied to this repository
- review comments and maintenance communication

## Enforcement

If you experience or witness unacceptable behavior, use the repository owner
contact path associated with this project.

This repository does not yet provide a dedicated conduct reporting portal, so
reports are handled on a best-effort basis through the existing maintainer
channel.

## Consequences

Maintainers may edit, remove, or reject content that violates this code of
conduct. Repeated or severe violations may result in loss of participation
privileges.
